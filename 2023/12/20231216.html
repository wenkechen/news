【2】根据 The Verge 的报道，字节跳动被OpenAI封杀！

因为字节也违反了 OpenAI 的服务条款，一直在使用GPT生成的数据在中国训练自己的竞争模型，从而违反了微软和OpenAI的开发人员许可证。字节跳动通过微软获取 OpenAI 的访问权限。

字节跳动的内部文件显示，他们在开发名为 Project Seed 的基础 LLM 时，几乎在每个阶段都依赖 OpenAI 的 API，包括用于训练和评估模型。由于过度依赖 OpenAI 的 API，Project Seed 的员工经常达到了 API 访问的最大限额。

涉及的员工清楚这一行为的含义，甚至讨论了如何通过“数据脱敏”来“洗白”证据。这种滥用如此普遍，以至于 Project Seed 的员工经常达到 API 访问的最大配额。

字节跳动回应称，GPT生成的数据已经在今年年中左右，从字节跳动的训练数据中删除。

【13】老友记钱德勒扮演者马修派瑞死因公布

洛杉矶县法医办公室称马修·派瑞死于 "氯胺酮（K粉）的急性作用"。验尸报告还指出，溺水、冠状动脉疾病和丁丙诺啡的影响也是导致死亡的原因。死亡方式认定为意外。

氯胺酮（K粉）在我国通常被视为毒品，洛杉矶县法医办公室发布了毒理学报告，派瑞在去世前一周半接受了氯胺酮输液治疗抑郁症和焦虑症。然而，他死亡时体内的氯胺酮“不可能因为输液治疗，氯胺酮的半衰期为3 至4小时甚至更短。”

马修·派瑞于10 月 28 日在其位于太平洋帕利塞德的家附近的Riviera乡村俱乐部打了两个小时的球后，被发现躺在按摩浴缸中，没有任何反应。急救人员接到心脏骤停电话后赶到现场，发现马修溺水身亡。

据法医称，马修·派瑞体内的氯胺酮导致了心血管过度刺激和呼吸抑制。但警方并没有在派瑞家中发现酒精、甲基苯丙胺、可卡因、海洛因、五氯苯酚或芬太尼的毒品证据。根据采访，马修已戒毒19个月。

尽管吸烟没有被列为派瑞死亡的促成因素，报告指出他每天吸两包烟，患有慢性阻塞性肺病、肺气肿和糖尿病。
